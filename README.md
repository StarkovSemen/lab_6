# Старков Семён ИТС-1 Лабораторная №6

# Задание 2-3

### Текст задачи

2)Вычитание переменной типа Time
(учесть, что возможен переход в
предыдущие сутки). Результат должен
быть типа Time.

3)Унарные операции:
++ добавление минуты к объекту типа Time.
-- вычитание минуты из объекта типа Time
Операции приведения типа:
int (неявная) – результатом является количество минут (время
переводится в минуты);
bool (явная) – результатом является true, если часы и минуты не равны
нулю, и falseв противном случае.
Бинарные операции:
< Timet1, Timet2 — время переводится в минуты, результатом является
true, если количество минут в левом операнде меньше, чем количество
минут в правом операнде, и false–в противном случае.
> Timet1, Timet2 — время переводится в минуты, результатом является
true, если количество минут в левом операнде больше, чем количество
минут в правом операнде, и false–в противном случае

### Алгоритм решения

**Класс `Time`**
   - Приватные поля: `_hours`, `_minutes` (тип `byte`).
   - Публичные свойства с проверкой (часы 0–23, минуты 0–59).
   - Конструкторы: по умолчанию (0:00), с параметрами, копирования.

2. **Метод `Subtract` (Задание 2)**
   - Переводит оба времени в минуты (`hours*60 + minutes`).
   - Вычисляет разность.
   - Если разность отрицательна, добавляет 1440 (24*60) – переход в предыдущие сутки.
   - Преобразует обратно в часы и минуты, возвращает новый объект `Time`.

3. **Перегрузка операторов (Задание 3)**
   - **Унарные `++` / `--`**:  
     - Общее количество минут увеличивается/уменьшается на 1.  
     - Выполняется нормализация.  
     - Возвращается новый объект `Time` (исходный не изменяется).
   - **Неявное приведение к `int`**:  
     - Возвращает `_hours * 60 + _minutes`.
   - **Явное приведение к `bool`**:  
     - Возвращает `(_hours != 0) || (_minutes != 0)`.
   - **Бинарные `<` и `>`**:  
     - Вычисляют общее количество минут для каждого операнда и сравнивают.

4. **Ввод данных**
   - Функция `ReadByte` с проверкой диапазона (от `min` до `max`). Использует `byte.TryParse` и цикл `do-while`.

5. **Тестирование в `Main()`**
   - Ввод двух времен, демонстрация вычитания (обычного и с переходом через сутки).
   - Проверка унарных `++` и `--` на граничных значениях (23:59 → 00:00 → 23:59).
   - Проверка приведения к `int` и `bool`.
   - Проверка бинарных операторов `<` и `>`.
